# 数据挖掘学习笔记与实践

> 基于吕欣、王梦宁老师编写的《数据挖掘》一书整理

## 关于本书

吕欣、王梦宁老师编写的《数据挖掘》是一部兼具理论深度与实践指导的优秀教材。本书系统地介绍了数据挖掘的核心概念、算法原理和实际应用，是数据科学领域学习者的宝贵资源。

### 本书特色

- **体系完整**：从数据预处理到集成学习，覆盖数据挖掘全流程
- **理论与实践结合**：每个知识点都配有详细的算法推导和实践案例
- **内容前沿**：涵盖XGBoost、LightGBM、t-SNE、UMAP等最新技术
- **应用导向**：结合金融、医疗、制造等行业实际案例

### 作者简介

- **吕欣**：数据挖掘领域资深学者，在机器学习和大数据分析方面有深入研究
- **王梦宁**：数据科学领域专家，致力于数据挖掘算法的理论研究与实践应用

## 技术栈

- **Python 3.x** - 主要编程语言
- **NumPy / Pandas** - 数据处理
- **SciPy / scikit-learn** - 统计分析与机器学习
- **Matplotlib / Seaborn** - 数据可视化
- **XGBoost / LightGBM** - 梯度提升算法
- **TensorFlow / Keras** - 深度学习

## 使用方法

```bash
# 克隆仓库
git clone <repository-url>

# 安装依赖
pip install numpy pandas scipy scikit-learn matplotlib seaborn xgboost lightgbm tensorflow
```
